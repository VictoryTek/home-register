# Release Notes

This directory contains detailed release notes for each version of Home Registry.

## Format

Release notes follow this naming convention:
- `vX.Y.Z.md` - Stable releases
- `vX.Y.Z-beta.N.md` - Beta pre-releases
- `vX.Y.Z-rc.N.md` - Release candidates
